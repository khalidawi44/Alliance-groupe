== Description ==

Thème WordPress pour mouvement militant, association, parti politique ou syndicat. Design impactant rouge et noir, typographie bold sans-serif (Anton + Inter).

Sections incluses : hero de mobilisation, manifeste, combats/thématiques, événements, recherche groupe local, actualités, formulaire de signature, paliers de don.

**Tous les textes sont des placeholders entre crochets `[ ]`** — à personnaliser pour chaque organisation.
