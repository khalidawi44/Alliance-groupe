Thème WordPress gratuit pour barbershops avec système de file d'attente QR code intégré.

== Description ==

AG Starter Barber est un thème WordPress conçu pour les barbershops et salons de coiffure urbains. Design sombre premium avec accents dorés, système de file d'attente par QR code intégré.

Fonctionnalités :

* Système de file d'attente QR code (les clients scannent et prennent un ticket)
* Calcul automatique du temps d'attente (basé sur le nombre de barbers et la durée moyenne)
* Tableau de bord admin pour gérer la file en temps réel
* Tarifs et prestations configurables
* Design responsive mobile-first
* Customizer pour personnaliser textes, prix, horaires
* Compatible avec AG Starter Companion pour l'import demo en 1 clic

== Installation ==

1. Téléchargez le fichier ZIP du thème
2. Dans wp-admin, allez dans Apparence > Thèmes > Ajouter > Téléverser
3. Activez le thème
4. Installez le plugin AG Starter Companion pour le contenu demo
5. Configurez vos prestations dans File d'attente > Réglages
