# AG Premium Barber

Pack **Premium** du thème AG Starter Barber (WordPress).

Tier intermédiaire entre **Free** (thème de base) et **Business** (Premium + 5 sections supplémentaires).

## Ce que Premium apporte (vs Free)

- **Typographie industrielle** : Bebas Neue (display uppercase XL) + Cormorant Garamond (italic chic) + Special Elite (typewriter vintage)
- **Palette vintage** : charbon `#0E0B0A` + accents rouge sang `#B12A1F` + or laiton `#B8924A` + crème vintage `#E8D9B0`
- **Hero override** : image background fixed parallax + voile sombre + titre Bebas Neue géant
- **Service cards** : look industriel sans border-radius, hover bordure rouge sang
- **File d'attente** : compteur Bebas Neue géant rouge sang + labels typewriter
- **Section titles** : Bebas Neue uppercase + accents italic Cormorant rouge

## Ce qui n'est PAS dans Premium (= seulement Business)

- Section **L'équipe** (4 barbiers avec photos)
- Section **La galerie** (masonry)
- Section **Témoignages** clients
- Section **Réservation** calendrier
- Section **Nous trouver** (adresse + horaires)
- Customizer Contact

## Activation

Le pack s'active si tier ≥ `premium` (donc aussi en mode `business`).

Body class ajoutée : `ag-bb-active` + `ag-bb-premium-active`.

## Mode test

Customizer → AG Premium Barber → Activation → coche "Mode test" pour prévisualiser sans licence Premium réelle.

## Convention CSS

Classes préfixées `ag-bb-*` (partagées avec Business pour éviter la duplication CSS). Aucune classe Free n'est surchargée.
