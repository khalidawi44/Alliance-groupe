== Description ==

Thème WordPress pour mouvement militant ou association politique. Design impactant rouge et noir, typographie bold sans-serif (Anton + Inter).

Sections incluses : hero de mobilisation, manifeste, combats/thématiques, événements, recherche groupe local, actualités, formulaire de signature, paliers de don.

**Tous les textes sont des placeholders entre crochets `[ ]`** — à personnaliser via le Customizer ou en éditant les fichiers du thème (en child theme idéalement).

== Personnalisation ==

WP Admin > Apparence > Personnaliser > Mouvement militant :
- Identité : nom, slogan, titre hero, sous-titre, label/URL CTA
- Contact : adresse, email, téléphone

Pour aller plus loin : créer un child theme et éditer `front-page.php` directement.
